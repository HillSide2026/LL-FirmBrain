# Project Plan

**Project:** Ontario Corp / Locarno
**Project ID:** HBP-015
**Stage:** Planning

## Planning Objective

Translate the HBP-015 charter into an execution-ready track selection and market-readiness process. Confirm FINTRAC standing, verify RPAA transfer posture, pursue EMI relationship, assess Track C, and assemble a buyer-ready package at the highest achievable track.

## Project Path

1. Lock Track A readiness — FINTRAC standing verified, AML documented, records clean
2. Pursue Track B — ConnectPay written indication, RPAA readiness, software posture
3. Assess Track C — sponsor-bank outreach and viability
4. Select sale track based on institutional build progress
5. Assemble market-ready package for selected track
6. Sell on ML1 direction

## Planning Workstreams

| Workstream | Objective | Primary Output |
|---|---|---|
| WS-01 Track A Readiness | Verify FINTRAC standing, document AML program, clean corporate records | Clean shell package ready for Track A sale at any time |
| WS-02 EMI Relationship | Obtain direct EMI written indication from ConnectPay; understand pre-revenue contracting terms | EMI written indication or documented assessment of feasibility |
| WS-03 RPAA Transfer Analysis | Confirm share-transfer implications, acquisition-of-control rules, re-registration risk | RPAA transfer memo for buyer diligence package |
| WS-04 Track B Build | If EMI obtained: ledger requirement, AML software (Rhizome default), Locarno product posture | Track B Preferred package — demo-ready compliance workflow |
| WS-05 Track C Assessment | Identify sponsor-bank candidates; assess willingness to engage pre-revenue entity | Track C viability note; pursue if feasible |
| WS-06 Market-Ready Package | Assemble buyer-readable diligence folder at achieved track level | Diligence index, management narrative, flow-of-funds, regulatory summary |
| WS-07 Scope Control | Prevent project from drifting into operating BaaS business | Scope reviews at each milestone |
