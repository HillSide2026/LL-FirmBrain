# Stakeholders

Project ID: LLP-027
Project Path: 04_INITIATIVES/LL_PORTFOLIO/08_MARKETING/LLP-014/LLP-027

## Core Stakeholders

- ML1: final authority on triage logic, SLA targets, response content, and
  practice area routing decisions; reviews any inquiry that falls outside
  defined triage categories
- ML2 / System: maintains inquiry logging, triage workflows, and response
  infrastructure within approved scope
- LLP-014 (Intake Management): parent project; receives cross-stage reporting
  and governs the handoff protocol that this project feeds

## Secondary Stakeholders

- LLP-026 (Lead Capture): upstream — web form submissions from lead capture
  are the primary source of logged inquiries
- LLP-028 (Consults): downstream — receives triaged and qualified inquiries
  via the handoff protocol; must align on what a complete handoff looks like
