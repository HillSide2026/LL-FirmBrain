# Success Criteria

The project is successful when:

1. `>=95%` of active matters include complete command metadata in generated artifacts.
2. Daily run emits all required outputs deterministically:
   - `MATTER_DIGEST.md`
   - Per-matter `MATTER_STATUS.md`
   - `INBOX_UNMAPPED.md`
3. Inbox-to-matter routing is label-first and citation-backed for every assignment.
4. All generated assertions include source pointers to Clio/SharePoint/Gmail evidence.
5. No ML2 artifact is treated as source truth in conflict with Clio/SharePoint/Gmail.
6. Slice 1 is operational with repeatable run outputs and auditable run state.
