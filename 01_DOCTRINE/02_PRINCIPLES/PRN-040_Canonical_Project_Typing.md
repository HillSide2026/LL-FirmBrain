# PRN-040 — Canonical Project Typing

Title: Canonical Project Typing

Statement:
Each project should be classified by its primary governance objective using one
canonical type. Mixed, aesthetic, or stage-driven labels should yield to the
single type that determines artifact, risk, and gate expectations.

Rationale:
Stable typing keeps governance predictable and prevents both over-governance and
under-governance.
